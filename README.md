# OPS102 Open Educational Resource

This repository contains the source files, based on the Seneca SCPA OER template (utilizing Docusaurus), for the Open Educational Resource (OER) for the Seneca Polytechnic OPS102 *Operating Systems for Programmers* course.

The URL for the public OER site is: [https://seneca-ictoer.github.io/OPS102/](https://seneca-ictoer.github.io/OPS102/)

To suggest a change to this content, please fork this repository, make whatever changes you are proposing, and submit a pull request. Alternately, you can open an issue in the GitHub issue tracker for this repository.
